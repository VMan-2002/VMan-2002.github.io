UI Style variables

These are the values for ui style's json file

three, two, one, go (String)

4 variables to use for countdown images.

threeSound, twoSound, oneSound, goSound (String)

4 variables to use for countdown sound effects.

numbers (Array<String>)

10 variables to use for the Combo numbers. Start from 0.

sick, good, bad, shit, sickcool (String)

Image files to use for ratings. "sickcool" is used for Sick rating while you have SFC.

combo (String)

Image file to use for the "Combo" graphic.

healthBar (String)

Image file for the health bar.

healthBarSides (Array<Float>)

How far inward the inner bar should be insetted in the health bar graphic. In order, the numbers are: the left side, top side, right side and bottom side.

healthBar (String)

Image file for the health bar.

comboScale, ratingScale, countdownScale (Float)

Scale for the combo numbers, ratings and countdown respectively

hudThingPos (Map<String, Array<Float>>)

Positioning for some HUD elements. The numbers are in order:

comboScale, ratingScale, countdownScale (Float)

Scale for the combo numbers, ratings and countdown respectively

iconEase (Float)

Multiplier for how fast the health icons should return to normal when bumping

iconEaseStr (String)

Name of an ease function to use for the health icon. Idk if this works

gameOverMusic, gameOverMusicEnd, pauseMusic (String)

Name of some music files
