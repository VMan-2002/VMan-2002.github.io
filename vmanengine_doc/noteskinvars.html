Noteskin variables

These are the values for noteskin's json file

image (String)

Image file to use for the noteskin.

imageDownscroll (String)

Image file to use for the noteskin when Scroll Direction is set to Downscroll.

scale (Float)

Rescale the notes

antialias (Bool)

Set to false for pixel style

arrows (Array<Animation>)

Array of the noteskin anims. If not supplied, use the default animations.
