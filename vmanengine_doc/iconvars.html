Health Icon variables

These are the values for health icon's json file, which is actually optional. Credits icons are placed in a subfolder of the icons folder named "credits". Animated health icons aren't fully tested, if they don't work, let me know

image (String)

Image file to use for the character icon. If an xml file exists with the same name, the health icon will be animated.

imagePlayer (String)

Image file to use for the character icon when playable.

initAnim (String)

Animation to play when the health icon is loaded.

antialias (Bool)

Set to false for pixel style

animations (Array<Animation>)

Array of the character's animations.

scale (Float)

Rescale the icon. Idk if this works

folderType (String)

Change the folder appearance when this icon is used as a folder icon.

position_freeplay (Array)

Reposition the icon specifically in Freeplay.
